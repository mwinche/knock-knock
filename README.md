## Getting Started

Install the module with: `npm install knock-knock`

```js
var knock-knock = require('knock-knock');
knock-knock.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g knock-knock
$ knock-knock --help
$ knock-knock --version
```
